⚙️ Синхронизация через GitHub Gist

Как настроить синхронизацию между устройствами:

  1. Создайте Personal Access Token на GitHub: Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Дайте права gist
  3. Вставьте токен ниже и нажмите "Сохранить"
  4. На других устройствах введите тот же токен — данные синхронизируются!

Данные хранятся в вашем приватном Gist на GitHub. Никто кроме вас не имеет к ним доступа.
